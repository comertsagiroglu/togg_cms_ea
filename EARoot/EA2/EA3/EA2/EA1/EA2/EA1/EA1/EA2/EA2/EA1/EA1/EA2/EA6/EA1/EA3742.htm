This function reads a value from a DPHY register space. A 32 bit read is performed. If the component is implemented in a smaller width, only the least significant data is read from the register. The most significant data will be read as 0. <br/>@param BaseAddress is the base address of the XCsiSs core instance.<br/>@param RegOffset is the register offset of the register (defined at the top of this file). <br/>@return The 32-bit value of the register. <br/>@note None.<br/>
