Run a self-test on the driver/device. Note this may be a destructive test if resets of the device are performed.  If the hardware system is not built correctly, this function may never return to the caller. <br/>@param   baseaddr_p is the base address of the MYVIDEOCRC instance to be worked on. <br/>@return  - XST_SUCCESS   if all self-test code passed - XST_FAILURE   if any self-test code failed <br/>@note    Caching must be turned off for this function to work.<br/>@note    Self test may fail if data memory and device are not on the same bus.<br/>
