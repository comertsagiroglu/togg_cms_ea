Notes: Receive the specified data from the device that has been previously addressed on the IIC bus. This function assumes the following: - The Rx Fifo occupancy depth has been set to its max. - Upon entry, the Rx Fifo is empty. - The 7 bit address has been sent. - The dynamic stop and number of bytes to receive has been written to Tx Fifo. <br/>@param BaseAddress contains the base address of the IIC Device.<br/>@param BufferPtr points to the buffer to hold the data that is received.<br/>@param ByteCount is the number of bytes to be received. The range of this value is greater than 0 and not higher than 255. <br/>@return The number of bytes remaining to be received. <br/>@note This function contains loops that could cause the function not to return if the hardware is not working.<br/>
