Interrupt Sources  Summary: Defines the data type for all the interrupt sources associated with the interrupt controller of the device.  Description: This data type can be used with interface functions to enable, disable, set, clear and to get status of any particular interrupt source.  Remarks: This data type is defined using the CMSIS data type that defines the interrupt sources set available.<br/>
