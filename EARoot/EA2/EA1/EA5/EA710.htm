Pre-conditions:<br/>- Power state = POWER_ON<br/>- Operational state = CMS_FAILURE<br/><br/>Scenario:<br/>- When the error condition is resolved the corresponding error flag is cleared.<br/>- Safety Observer function checks the error flags periodically and when all error flags are clear exits the system from CMS_FAILURE state and enables the LCD backlight.<br/>-  Error conditions:<br/>    <font color="#0000c0"><i>SAFE_STATE_FROZEN_FRAME</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_DELAYED_FRAME</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_FLIPPED_FRAME</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_SYS_CHECK</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_ERRB</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_CSI</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_MAINTENANCE_MODE</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_CAM_QUEUE</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_I2C</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_LED_OUT</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_DRIVER</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_SHORT_GND</i></font>,<br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_SHORT_LED</i></font><br/>    <font color="#0000c0"><i>SAFE_STATE_BACKLIGHT_EXT_PIN</i></font><br/><font color="#0000c0"><i>    SAFE_STATE_LCD</i></font><br/>- Frame error handler sets the operational state to PRIMARY_OPERATION and returns it to IDCC when it is queried.<br/>- IDCC  operational mode query period is  100msec.<br/>- Safety Observer period is  300msec.<br/><br/>Post-conditions:<br/>Operational state = PRIMARY_OPERATION (FREEWAY /PARKING / LANE_CHANGE)<br/>LCD backlight on<br/><br/>
