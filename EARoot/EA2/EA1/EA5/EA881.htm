Pre-conditions:<br/>- Camera serializer and Display deserializer are locked.<br/>- Image Sensor is in streaming state<br/>- Power state = POWER_ON<br/>- Operational state = PRIMARY_OPERATION<br/><br/>Scenario:<br/>- When one of the frame error conditions (frozen image, delayed image, orientation failure) is detected by the FPGA hardware, frame error event is sent to the display firmware.<br/>- Frame error handler sets the operational state to CMS_FAILURE and returns it to IDCC when it is queried.<br/>- Frame error handler disables the LCD backlight<br/>- Display fw sets the related diagnostic code and returns it to IDCC when it is queried.<br/>- IDCC  diagnostic data query period is  100msec.<br/>- IDCC  operational mode query period is  100msec.<br/><br/>Post-conditions:<br/>Operational state = CMS_FAILURE<br/>LCD backlight off<br/>
