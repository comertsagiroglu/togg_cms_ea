SW Purpose: The purpose of SW function “<b>Backlight_status</b>” of the SW module “<b>devices</b>” of the SW component “<b>CMS_SW</b>” is to check the following registers of backlight controller: (AND)<br/><ul>
<li>Check backlight I2C connection -#gt; Read device id (<b>0x24</b>) from <b>Dev_ID (0x00) register</b> AND</li><li>Check LCD LED output enable status. Error if not enabled -#gt; <b>ISET (0x02) register ENA bit.5</b> AND</li><li>Check if LCD is connected -#gt; <b>OPEN (0x1B) register bit.1-0</b> AND</li><li>Check short to ground error -#gt; <b>SHORTGND register (0x1C) bit.1-0</b> AND</li><li>Check short between LED outputs -#gt; <b>SHORTED_LED register (0x1D) bit.1-0</b> AND</li><li>Check backlight driver external pin -#gt; <b>GPIO_0::GPIO_CHAN_1</b> clear</li></ul>
