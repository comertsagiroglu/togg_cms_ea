SW Purpose: The purpose of SW function “<b>ConfigDisplayBacklight</b>” of the SW module “<b>devices</b>” of the SW component “<b>CMS_SW</b>” is to de-initialize the following registers of backlight controller with the following values (AND)<br/><ul>
<li>Enable -#gt; Set bit.5 of <b>ISET register (0x02)</b> AND</li><li>Disable DIM pin -#gt; Clear bit.3 of <b>IMODE register (0x03)</b> AND</li><li>Disable hybrid dimming -#gt; Clear bit.2 of <b>IMODE register (0x03)</b> AND</li><li>Set channel 1 on time to 32.768 msec -#gt; <b>TON1H (0x04)</b> + <b>TON1L (0x05)</b> + <b>TON1LSB (0x0C bit.1-0)</b> = 0xFFFF AND</li><li>Set channel 2 on time to 32.768 msec -#gt; <b>TON2H (0x06)</b> + <b>TON2L (0x07)</b> + <b>TON2LSB (0x0C bit.3-2)</b> = 0xFFFF</li></ul>
