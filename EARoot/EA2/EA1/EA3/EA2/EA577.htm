When the SW component “<b>CMS_SW</b>” makes the transition from OPERATIONAL_MODE_LANE_CHANGE to OPERATIONAL_MODE_LANE_FREEWAY, the SW component “<b>CMS_SW</b>” shall call the SW function “<b>ih_OverlayStatus</b>” of the SW module “<b>idcc_handler</b>” of the SW component “<b>CMS_SW</b>” which shall perform the following functionality: (OR)<br/><ul>
<li>If the mirror position variable <i>gMirrorPosition</i> is set to MIRROR_POS_DRIVER (AND), <ul>
<li>Clear Lane Change FoV Left Overlay AND</li><li>Set Left Mirror Freeway FoV resolution x and y coordinates (x0: 187, x1: 939, y0: 146, y1: 596) OR</li></ul>
</li><br/><li>If the mirror position variable <i>gMirrorPosition</i> is set to MIRROR_POS_PASSENGER (AND), <ul>
<li>Clear Lane Change FoV Right Overlay AND</li><li>Set Right Mirror Freeway FoV resolution x and y coordinates (x0: 297, x1: 1067, y0: 132, y1: 593)</li></ul>
</li><br/></ul>
