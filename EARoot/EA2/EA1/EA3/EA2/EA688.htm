SW Purpose: The purpose of the SW function “<b>ConfigCSIDES</b>” of the SW module “<b>devices</b>” of the SW component “<b>CMS_SW_FastInit</b>” is to initialize the following registers of the display deserializer (MAX9296B) with the following values: (AND)<br/>• Set mapping enable low byte: Enable: MAP_SRC_0, MASP_DST_0; MAP_SRC_1, MASP_DST_1; MAP_SRC_2, MASP_DST_2 -#gt; Write <b>0x07</b> to <b>MIPI_0_TX11 register (0x40B)</b> AND<br/>• Set CSI2 controller destination for MAP_SRC_2 and MAP_DST_0 to DPHY1, MAP_SRC_1 and MAP_DST_0 to DPHY1, MAP_SRC_0 and MAP_DST_0 to DPHY1 -#gt; Write <b>0x15</b> to <b>MIPI_0_TX45 register (0x42D)</b> AND<br/>• Set VC and DT for source mapping 0: Virtual channel 0, Data type YUV422 -#gt; Write <b>0x1E</b> to <b>MIPI_0_TX13 register (0x40D)</b> AND<br/>• Set VC and DT for destination mapping 0: Virtual channel 0, Data type YUV422 -#gt; Write <b>0x1E</b> to <b>MIPI_0_TX14 register (0x40E)</b> AND<br/>• Select 2 data lanes -#gt; Write <b>0x40</b> to <b>MIPI_1_TX14 register (0x44A)</b> AND<br/>• Set MIPI PHY Port A Lane Mapping: Map PHY1 D0 to D0, PHY1 D1 to D1, PHY0 D0 to D2, PHY0 D1 to D3 (default) -#gt; Write <b>0x4E</b> to <b>MIPI_PHY3 register (0x333)</b> AND<br/>• Set MIPI PHY Port B Lane Mapping: Map PHY3 D0 to D2, PHY3 D1 to D3, PHY2 D0 to D0, PHY2 D1 to D1 (default) -#gt; Write <b>0x4E</b> to <b>MIPI_PHY4 register (0x334)</b> AND<br/>• Set CSI PHY1 output frequency to 5 x 100MHz = 500MHz, Software override for frequency fine tuning disabled -#gt; Write <b>0x25</b> to <b>BACKTOP_25 register (0x320)</b> AND<br/>• Set stream id for receiving to 2, No CRC on receive packets -#gt; Write <b>0x02</b> to <b>RX0 register (0x50)</b> AND<br/>• Enable: LOCK output, ERRB output, Power-up line fault monitor 3, Power-up line fault monitor 2, Power-up line fault monitor 1, Power-up line fault monitor 0 -#gt; Write <b>0xFF</b> to <b>REG5 register (0x5)</b> AND<br/>• Enable reporting of; PHY B interrupt, PHY A interrupt, remote error status, memory error status, idle word errors, decoding errors at ERRB pin -#gt; Write <b>0xF7</b> to <b>INTR2 register (0x1A)</b> AND<br/>• Enable reporting of; eye-opening minitor error, combined ARQ maximum retransmission limit error, combined ARQ retransmission event, packet count, watermark errors at ERRB pin -#gt; Write <b>0xCF</b> to <b>INTR4 register (0x1C)</b> AND<br/>• Enable reporting of; VDDCMP interrupt, PORTZ interrupt, VDDBAD interrupt, frame synch errors, video line CRC errors, PRBS errors, Video pixel CRC error counter interrupt at ERRB pin -#gt; Write <b>0xFD</b> to <b>INTR6 register (0x1E)</b> AND<br/>• Enable reporting of GMSL1 Link A and Link B errors at ERRB pin -#gt; Write <b>0xFF</b> to <b>GMSL1_ERR_OEN register (0xF03)</b> AND<br/>• Enable video line CRC -#gt; Write 0X02 to RX0 register (0x100) AND<br/>• Set automatically reset DEC_ERR_A, DEC_ERR_B and IDLE_ERR registers after ERRB pin is asserted for 1us -#gt; Set <b>bit.3</b> of <b>INTR0 register (0x18)</b> AND<br/>• Enable automatic link configuration -#gt; Set <b>bit.4</b> of <b>CTRL0 register (0x10)</b> AND<br/>• One-shot reset -#gt; Set <b>bit.5</b> of <b>CTRL0 register (0x10</b>)<br/>(QM)<br/><br/>
