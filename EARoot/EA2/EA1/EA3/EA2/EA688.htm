SW Purpose: The purpose of the SW function “<b>ConfigSER</b>” of the SW module “<b>devices</b>” of the SW component “<b>CMS_SW_FastInit</b>” is to initialize the following registers of the camera serializer (MAX9295E) with the following values: (AND)<br/>• Enable LOCK/ERRB pin output -#gt; Set <b>bit.6</b> and <b>bit.7</b> of <b>REG5 register (0x05)</b> AND<br/>• Set MIPI_Rx mode 1x4 (4 data lanes) -#gt; Set <b>bit.6</b> and <b>bit.0</b> of <b>MIPI_RX0 register (0x330)</b> AND<br/>• Enable CSI mode -#gt; Set <b>bit.7</b> of <b>MIPI_RX0 register (0x330)</b> AND<br/>• Set MIPI Rx two data lanes for Port A -#gt; Write <b>0x04</b> to <b>MIPI_RX1 register (0x331)</b> AND<br/>• Set PHY1 lane mapping as: Map PHY1 D0 to MIPI 2 AND Map PHY1 D1 to MIPI 3 AND -#gt; Write <b>0x4E</b> to <b>MIPI_RX2 register (0x332)</b> AND<br/>• Set PHY2 lane mapping as: Map PHY1 D1 to MIPI 1 AND Map PHY1 D0 to MIPI 0 -#gt; Write <b>0xE4</b> to <b>MIPI_RX3 register (0x333)</b> AND<br/>• Set YUV422 data type for Pipeline Z -#gt; Write <b>0x1E</b> to <b>FRONTTOP_16 register (0x318)</b> AND<br/>• Enable all video pipes -#gt; Write <b>0x0F</b> to <b>FRONTTOP_9 register (0x311)</b> AND<br/>• Enable line start information frames and CSI Port A -#gt; Write <b>0x50</b> to <b>FRONTTOP_0 register (0x308)</b> AND <br/>• Enable all video pipes -#gt; Write <b>0xF3</b> to <b>REG2 register (0x2)</b> AND<br/>• Set normal polarity for all data lanes -#gt; Write <b>0x80</b> to <b>MIPI_RX4 register (0x334)</b> AND<br/>• Enable reporting of; remote error status, line-fault interrupt, idle-word errors, decoding errors, at ERRB pin -#gt; Write <b>0x3F</b> to <b>INTR2 register (0x1A)</b> AND <br/>• Enable reporting of; eye-opening monitor error for Link B, eye-opening error for Link A, over-voltage status, combined ARQ maximum retransmission limit error flag, package count flag, watermark errors, at ERRB pin -#gt; Write <b>0xEF</b> to <b>INTR4 register (0x1C)</b> AND<br/>• Enable reporting of; VDD comparator output, PORTZ interrupt, VDDBAD interrupt, MIPI Rx errors, at ERRB pin -#gt; Write <b>0xE1</b> to <b>INTR6 register (0x1E)</b> AND<br/>• Enable automatically resetting of DEC_ERR_A and IDLE_ERR bit-fields after ERRB pin is asserted for 1us -#gt; Set <b>bit.3</b> of <b>INTR0 register (0x18)</b> AND<br/>• Activate LINK A -#gt; Set <b>bit.0</b> of <b>CTRL0 register (0x10)</b> AND<br/>• One-shot reset -#gt; Set <b>bit.5</b> of <b>CTRL0 register (0x10)</b><br/>(QM)<br/><br/>
