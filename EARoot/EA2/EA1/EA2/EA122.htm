SW Purpose: The purpose of sw module ‘CsiRx_status’ is to check the CSI Receiver Core Frame Error via checking the following registers of the FPGA Csi-Rx Core: (AND) (ASIL B)<br/><ul>
<li>Check Csi-Rx core is enabled. Error if not enabled -#gt; Csi-Rx Core Config Register (CsiRxSs.Config.BaseAddr, XCSI_CCR_OFFSET, mask=0x3, shift=0x0). Error if not 1 AND</li><li>Check active number of lanes. Error if not 2  -#gt; Csi-Rx Core Protocol Config Register (CsiRxSs.Config.BaseAddr, XCSI_PCR_OFFSET, mask=0x1F, shift=0x0) bit.1-0. Error if not 1 AND</li><li>Check max number of lanes. Error if not 2  -#gt; Csi-Rx Core Protocol Config Register (CsiRxSs.Config.BaseAddr, XCSI_PCR_OFFSET, mask=0x1F, shift=0x0) bit.4-3. Error if not 8 AND</li><li>Check Csi-Rx frame reception. Error if not received -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.31. Error if 0 AND</li><li>Check Csi-Rx interrupt status. Error if flags set -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.30-0. Error if not 0 AND</li><li>Check Csi-Rx VCX frame error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.30. Error if 1 AND</li><li>Check Csi-Rx VC0 frame level error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.0. Error if 1 AND</li><li>Check Csi-Rx VC0 frame synch error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.1. Error if 1 AND</li><li>Check Csi-Rx YUV420 WC error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.28. Error if 1 AND</li><li>Check Csi-Rx incorrect lane config error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.21. Error if 1 AND</li><li>Check Csi-Rx SoT error. Error if active -#gt; Csi-Rx ISR Status Register (CsiRxSs.Config.BaseAddr, XCSI_ISR_OFFSET, XCSI_ISR_ALLINTR_MASK & ~(XCSI_ISR_STOP_MASK), XCSI_ISR_VC0FLVLERR_SHIFT) bit.13. Error if 1 AND</li><li>Check Csi-Rx VCX frame error. Error if active -#gt; Csi-Rx VCX Frame Error Register (CsiRxSs.Config.BaseAddr, XCSI_VCX_FE_OFFSET, XCSI_INTR_VCFE_MASK, shift=0x0). Error if not 0 AND</li><li>Check Csi-Rx lane-0 SOT error. Error if active -#gt; Csi-Rx Lane-0 Info Register (CsiRxSs.Config.BaseAddr, XCSI_L0INFR_OFFSET, XCSI_LXINFR_SOTSYNCERR_MASK | XCSI_LXINFR_SOTERR_MASK, 0x0). Error if 1 AND</li><li>Check Csi-Rx lane-1 SOT error. Error if active -#gt; Csi-Rx Lane-0 Info Register (CsiRxSs.Config.BaseAddr, XCSI_L1INFR_OFFSET, XCSI_LXINFR_SOTSYNCERR_MASK | XCSI_LXINFR_SOTERR_MASK, 0x0). Error if 1 AND</li><li>Check Csi-Rx VC0 info1 error. Error if active -#gt; Csi-Rx VC0 Info1 Register (CsiRxSs.Config.BaseAddr, XCSI_VC0INF1R_OFFSET, 0xFFFFFFFF, 0x0). Error if 0 AND</li><li>Check Csi-Rx VC0 info2 error. Error if active -#gt; Csi-Rx VC0 Info2 Register (CsiRxSs.Config.BaseAddr, XCSI_VC0INF2R_OFFSET, 0xFFFFFFFF, 0x0). Error if 0 AND</li><li>Check Csi-Rx D-Phy clock lane control error. Error if active -#gt; Csi-Rx D-Phy Clock Lane Control Register (CsiRxSs.Config.BaseAddr, 0x8 + XDPHY_CLSTATUS_REG_OFFSET, XDPHY_CLSTATUS_ALLMASK, 0x0) bit.5. Error if 1 AND</li></ul>
