SW: The sw function <b>LCD_Status</b> calls the sw function <b>sm_LCDErrorHandler</b> immediately when at least one of the following error flags is set: (OR)<br/><ul>
<li>bit.0: LVDS Lock failure: No LVDS clock OR</li><li>bit.1: GAS function failure: GAS (Gate All Select) Any one of the following cases triggers the GAS function (OR) OR<ul>
<li>VCC lower than 2.5V  OR</li><li>VSP lower than +4V OR</li><li>VSN higher than -4V OR</li></ul>
</li><br/></ul>
<ul>
<li>bit.2: Internal source circuit failure: Source IC internal LD Signal (Latch Data Signal fail) fail OR</li><li>bit.3: Tradition gate signal failure: STV signal fail OR</li><li>bit.4: Self-protection mode failure: No video (No LVDS clock) OR</li><li>bit.5: EEPROM reload failure: Unused OR</li><li>bit.6: OTP trimming fault: Internal used for Hannstar OR</li><li>bit.7: PFM abnormal fault: VSP/VSN power fail OR</li><li>bit.8: Glass crack fault: Glass crack detect function is not available OR</li><li>bit.9: OTP reload fault: OTP reload fail. Driver IC will compare the reload value with one of the default register value. The fail flag will be high.</li></ul>
