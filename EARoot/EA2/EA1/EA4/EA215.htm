Pre-conditions:<br/>- Power state = POWER_ON<br/>- Operational state = PRIMARY_OPERATION - PARKING<br/><br/>Scenario:<br/>- The display must be in the parking operational state i.e. the vehicle must be stationary in order for the display to make the maintenance mode state transition.<br/>- IDCC sends set_maintenance_mode message with one of the mode attributes:<br/>    Fw Update - Camera MCU FW<br/>    Fw Update - Camera Sensor FW<br/>    Fw Update - Display FPGA HW<br/>    Fw Update - Display FPGA FW<br/>    Fw Update - Camera MCU BL<br/>    Zoom Change <br/>- Display maintenance mode handler disables the LCD backlight.<br/>- Display maintenance mode handler sets the operational state to maintenance mode.<br/><br/>Post-conditions:<br/>Operational state = MAINTENANCE<br/>LCD backlight off<br/><br/>
