Main Routine Codes Enumeration  Summary: Defines return codes for "main".  Description: This enumeration provides a predefined list of return codes for the main function.  These codes can be passed into the MAIN_RETURN_CODE macro to convert them to the appropriate type (or discard them if not needed) for the Microchip C-language compiler in use.  Example:<br/>#lt;code#gt; MAIN_RETURN main ( void ) { SYS_Initialize(...);  while(true) { SYS_Tasks(); }  return MAIN_RETURN_CODE(MAIN_RETURN_SUCCESS); }<br/>#lt;/code#gt;  Remarks: The main function return type may change, depending upon which family of Microchip microcontrollers is chosen. Refer to the user documentation for the C-language compiler in use for more information.<br/>
