System Module Init  Summary: Initializes a module (including device drivers) as requested by the system.  Description: This structure provides the necessary data to initialize or reinitialize a module (including device drivers). The structure can be extended in a module specific way as to carry module specific initialization data.  Remarks: This structure is used in the device driver routines DRV_#lt;module#gt;_Initialize and DRV_#lt;module#gt;_Reinitialize that are defined by each device driver.<br/>
