I2C Driver Transfer Events  Summary: Identifies the possible events that can result from a buffer add request.  Description: This enumeration identifies the possible events that can result from a buffer add request caused by the client calling either the DRV_I2C_ReadTransferAdd, DRV_I2C_WriteTransferAdd or DRV_I2C_WriteReadTransferAdd functions.  Remarks: One of these values is passed in the "event" parameter of the event handling callback function that the client registered with the driver by calling the DRV_I2C_TransferEventHandlerSet function when a buffer transfer request is completed.<br/>
