Driver Client Status  Summary: Identifies the current status/state of a client's connection to a driver.  Description: This enumeration identifies the current status/state of a client's link to a driver.  Remarks: The enumeration used as the return type for the client-level status routines defined by each device driver or system module (for example, DRV_USART_ClientStatus) must be based on the values in this enumeration.<br/>
